!box: 基本操作
    * **起動**: python editor.py (※事前に `python server.py` を別窓で起動しておくこと)
    * **開く**: ツールバーの「開く」ボタンから `.txt` ファイルを選択。
    * **保存**: `Ctrl + S` またはツールバーの「保存」ボタン。
    * **プレビュー**: 「プレビューウィンドウ起動」ボタンで専用ブラウザを表示。保存するたびに自動更新される。

# 入力支援ショートカット

瀬川 左
    執筆を効率化するためのショートカットキーを用意したわ。
    これを覚えるとマウスに触らずに記事が書けるようになるわよ。

!box: ショートカット一覧
    * `Ctrl + S` : 上書き保存 & プレビュー更新
    * `Ctrl + B` : 囲み枠 (`!box:`) を挿入
    * `Ctrl + M` : 数式ブロック (`!math:`) を挿入
    * `Ctrl + I` : 画像 (`!img:`) を挿入（ファイル選択）
    * `Ctrl + K` : コードブロック (`!code:`) を挿入
    * `Ctrl + F` : 折りたたみ (`!fold:`) を挿入
    * `Tab` : インデント（半角スペース4つ）を入力

# 画像の挿入方法

栃村 右
    画像を貼りたいときはどうすればいいんですか？

瀬川 左
    とっても簡単よ。以下の2つの方法があるわ。

!box: 画像インポート
    1. **ドラッグ＆ドロップ**: 
       エディタ画面に画像ファイルを直接ドラッグ＆ドロップする。
    2. **ショートカット**: 
       `Ctrl + I` を押してファイル選択ダイアログから選ぶ。
    どちらの方法でも、自動的に `sources/ジャンル/` フォルダに画像がコピーされ、記事内にコマンドが挿入されるわ。
    ※ 事前に記事ファイルを保存して、ジャンル（保存場所）を確定させておく必要があるわよ。

# 記述ルール (Segawa Script)

!box: 会話形式
    話者名の行の次に、インデントして内容を書く。
    空行でブロック終了。

!box: 囲み枠 (!box)
    `!box: タイトル`
    タイトルの文字によって色が自動で変わるわ。
    * 「定義」「Def」 -> 緑
    * 「定理」「Thm」 -> 青
    * 「証明」「Proof」 -> グレー
    * 「まとめ」「宿題」 -> 特別なデザイン

!code: text
    !box: 定義 1.1
        ここは定義の本文。

!box: 数式 (!math)
    MathJax を使って TeX 記法で数式が書けるわ。
    インラインは `$ ... $`、ディスプレイは `$$ ... $$` よ。

!math:
    $$ e^{i\pi} + 1 = 0 $$

!box: コード (!code)
    プログラミングコードをハイライト表示するわ。

!code: python
    print("Hello, Segawa Lab!")
