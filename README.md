# Segawa Studio Generated Site

このリポジトリは、**Segawa Studio** によって生成された対話形式の技術ブログ/ゼミ記録サイトです。

## サイトの構成
- **Top**: サイトの概要と最新の日誌が表示されます。
- **Seminars**: 過去の記事がジャンルごとに分類されています。
- **Articles**: 会話劇形式で解説が進む技術記事です。

## 技術仕様
- **変換エンジン**: Segawa Script -> JSON -> Dynamic HTML
- **フロントエンド**: 
  - [MathJax](https://www.mathjax.org/) (数式レンダリング)
  - [Highlight.js](https://highlightjs.org/) (コードハイライト)
- **ホスティング**: GitHub Pages 推奨
